# 核心财务指标

## 盈利能力

| 指标 | 公式 | 说明 |
| :--- | :--- | :--- |
| 毛利率 | `(营业收入 - 营业成本) / 营业收入` | 产品竞争力与成本压力 |
| 净利率 | `净利润 / 营业收入` | 最终盈利能力 |
| ROE | `净利润 / 平均净资产` | 股东权益回报 |
| 扣非/净利润 | `扣非净利润 / 净利润` | 核心经营利润占比 |

## 偿债与资产质量

| 指标 | 公式 | 说明 |
| :--- | :--- | :--- |
| 资产负债率 | `总负债 / 总资产` | 财务杠杆和偿债压力 |
| 流动比率 | `流动资产 / 流动负债` | 短期偿债能力 |
| 速动比率 | `(流动资产 - 存货) / 流动负债` | 剔除存货后的短期偿债能力 |
| 有息负债率 | `有息负债 / 总资产` | 债务压力 |

## 现金流与盈利质量

| 指标 | 公式 | 说明 |
| :--- | :--- | :--- |
| 净现比 | `经营现金流 / 净利润` | 利润现金含量 |
| 应计利润率 | `(净利润 - 经营现金流) / 总资产` | 高于 10% 需关注利润质量 |
| 应收/营收 | `应收账款 / 营业收入` | 回款质量 |
| 存货/营收 | `存货 / 营业收入` | 库存压力 |

## 杜邦三因子

`ROE = 净利率 * 资产周转率 * 权益乘数`

- 净利率：盈利能力驱动。
- 资产周转率：运营效率驱动。
- 权益乘数：杠杆驱动，提升 ROE 的同时也可能提升风险。
