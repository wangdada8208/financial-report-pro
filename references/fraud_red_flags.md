# 财务红旗规则

## 红旗信号

| 红旗 | 检测方法 | 严重度 |
| :--- | :--- | :--- |
| 存贷双高 | 货币资金和有息负债同时高于营收 30% | 高 |
| 应收暴增 | 应收增速明显高于营收增速 | 高 |
| 存货异常 | 存货/营收超过 50% 或快速上升 | 高 |
| 经营现金流为负 | 净利润为正但 CFO 为负 | 高 |
| 关联交易占比高 | 关联交易/营收超过 30%，需人工从附注确认 | 高 |
| 频繁更换审计师 | 三年内多次更换，需人工确认 | 中 |
| 在建工程不转固 | 在建工程/固定资产持续偏高，需人工确认 | 中 |
| 预付账款异常 | 预付/营收突然上升，需人工确认 | 中 |
| 少数股东损益异常 | 少数股东损益/净利润波动大，需人工确认 | 中 |
| 非标审计意见 | 保留、否定、无法表示意见 | 高 |
| 研发资本化率过高 | 研发资本化/研发总额超过 50% | 中 |
| 商誉占比高 | 商誉/净资产超过 30% | 中 |

## 风险等级

- 0-1 个红旗：低。
- 2-3 个红旗：中，需要深入核查。
- 4-5 个红旗：高，建议回避或等待更多证据。
- 6 个及以上：极高，强烈回避。
