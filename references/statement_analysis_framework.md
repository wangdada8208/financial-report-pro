# 三表分析框架

## 核心顺序

1. 先看利润表：收入、成本、毛利率、净利率、扣非净利润。
2. 再看资产负债表：现金、应收、存货、商誉、有息负债、权益。
3. 最后用现金流验证利润：经营现金流是否覆盖净利润。

## 三表勾稽

- `资产 = 负债 + 所有者权益`，差异过大时优先怀疑提取错误或口径不一致。
- 长期看 `净利润` 应与 `经营现金流` 同方向，若利润为正但 CFO 为负，应关注收入确认、应收账款和存货。
- 若有期初期末现金数据，可进一步验证 `期末现金 = 期初现金 + CFO + CFI + CFF`。

## 盈利质量评分

| 指标 | 优秀 | 一般 | 较差 | 权重 |
| :--- | :--- | :--- | :--- | :--- |
| CFO/净利润 | `>1.2` | `0.8-1.2` | `<0.8` | 25% |
| 应收增速 vs 营收 | 应收更慢 | 接近同步 | 应收明显更快 | 20% |
| 扣非/净利润 | `>90%` | `70%-90%` | `<70%` | 20% |
| 经营现金流趋势 | 增长 | 小幅下滑 | 明显下滑 | 20% |
| 存货增长压力 | 放缓 | 温和增长 | 快速增长 | 15% |

综合评分大于等于 2.5 为优秀，1.5 到 2.5 为需要关注，低于 1.5 为盈利质量差。

## 现金流质量矩阵

| CFO | CFI | CFF | 状态 |
| :--- | :--- | :--- | :--- |
| + | - | - | 优秀：经营造血、投资扩张、还债或分红 |
| + | - | + | 扩张：经营造血，融资加速投入 |
| + | + | - | 稳健：经营现金为正，回收投资或压缩资产 |
| - | - | + | 危险：经营失血，依赖融资扩张 |
| - | + | + | 困境：卖资产和融资维持现金 |
| - | + | - | 衰退：卖资产还债，经营现金流为负 |
